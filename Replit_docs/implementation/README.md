# Implémentation

Plans et rapports d'implémentation de fonctionnalités.

## Convention de nommage

```
YYYY-MM-DD__DOMAIN__TOPIC__PLAN.md
YYYY-MM-DD__DOMAIN__TOPIC__REPORT.md
```

## Exemples

```
2026-01-15__PAYMENTS__option_a_upgrade_flow__PLAN.md
2026-01-20__ONBOARDING__simplified_enrollment__REPORT.md
```

## Contenu attendu

Pour un PLAN:
- Objectif
- Spécifications
- Étapes d'implémentation
- Critères de succès

Pour un REPORT:
- Résumé
- Implémentation réalisée
- Tests effectués
- Résultats
