# Sécurité

Documentation de sécurité : audits, politiques, rapports.

## Sous-dossiers

- `audits/` - Audits de sécurité
- `policies/` - Politiques de sécurité
- `reports/` - Rapports de vulnérabilités

## Convention de nommage

```
YYYY-MM-DD__SECURITY__TOPIC__DOC_TYPE.md
```

## Exemples

```
2026-01-20__SECURITY__node_vulnerabilities__AUDIT.md
2026-01-15__SECURITY__data_encryption__POLICY.md
```
