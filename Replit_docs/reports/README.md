# Rapports

Documents synthétisant les résultats d'une implémentation, correction ou analyse.

## Convention de nommage

```
YYYY-MM-DD__DOMAIN__TOPIC__REPORT.md
```

## Exemples

```
2026-01-21__ROLES__backoffice_role_simplification__REPORT.md
2026-01-20__PAYMENTS__membership_plans_500_fix__REPORT.md
```

## Contenu attendu

- Résumé exécutif
- Contexte / problème initial
- Actions réalisées
- Résultats / métriques
- Prochaines étapes (si applicable)
