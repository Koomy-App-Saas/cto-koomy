# Décisions (ADR)

Architecture Decision Records - Décisions techniques ou produit importantes.

## Convention de nommage

```
YYYY-MM-DD__DOMAIN__TOPIC__DECISION.md
```

## Exemples

```
2026-01-15__DATA__users_accounts_separation__DECISION.md
2026-01-10__PAYMENTS__option_a_payment_flow__DECISION.md
```

## Contenu attendu

- Contexte
- Décision prise
- Alternatives considérées
- Conséquences
- Statut (proposé/accepté/déprécié)
