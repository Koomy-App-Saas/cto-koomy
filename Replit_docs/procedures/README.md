# Procédures & Runbooks

Procédures opérationnelles et runbooks.

## Convention de nommage

```
YYYY-MM-DD__DOMAIN__TOPIC__PROCEDURE.md
YYYY-MM-DD__DOMAIN__TOPIC__RUNBOOK.md
```

## Exemples

```
2026-01-10__INFRA__deployment_release__PROCEDURE.md
2026-01-05__SECURITY__weekly_checkup__RUNBOOK.md
```

## Contenu attendu

- Objectif
- Prérequis
- Étapes détaillées
- Vérifications
- Rollback (si applicable)
