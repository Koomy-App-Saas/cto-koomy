# Incidents

Postmortems et analyses d'incidents de production.

## Convention de nommage

```
YYYY-MM-DD__DOMAIN__TOPIC__INCIDENT.md
```

## Exemples

```
2026-01-21__AUTH__admin_login_500__INCIDENT.md
2026-01-18__PAYMENTS__stripe_webhook_failure__INCIDENT.md
```

## Contenu attendu

- Résumé de l'incident
- Timeline
- Cause racine
- Impact
- Actions correctives
- Leçons apprises
